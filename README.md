# ac3-algorithm

This is project resolving Sudoku problem using AC3 algorithm

# Running

$ pip install tkinter

$ pip install pilow

$ app.py

- Step 1: Create data in input.txt file 

- Step 2: Click AND-OR graph search button to find solution
